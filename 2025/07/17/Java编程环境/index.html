安装Java开发工具包

下载JDK

原先，Oracle公司会提供最新、最完备的Java开发工具包（JDK）版本。如今，很多不同公司（包括Microsoft.Amazon、RedHat和Azul）都提供了最新的OpenJDK构建版本，有些公司的许可条件比Oracle公司更宽松。

  1. 访问 Oracle JDK 下载页面。
  2. 选择适合你操作系统的版本（Windows、macOS、Linux）。
  3. 下载并运行安装程序。

设置JDK

  1. 安装完成后，设置环境变量：
    • Windows：右键“此电脑” -> “属性” -> “高级系统设置” -> “环境变量”，添加 JAVA_HOME 指向 JDK 安装目录。
    • macOS/Linux：在 ~/.bash_profile 或 ~/.bashrc 中添加 export JAVA_HOME=/path/to/jdk。
  2. 更新 PATH 变量，添加 JDK 的 bin 目录：
    • Windows：在环境变量中编辑 Path，添加 %JAVA_HOME%\bin。
    • macOS/Linux：在 ~/.bash_profile 或 ~/.bashrc 中添加 export PATH=$JAVA_HOME/bin:$PATH。
  3. 验证安装：
    • 打开命令行，输入 java -version，应显示 JDK 版本信息。
    • 输入 javac -version，应显示编译器版本信息。

安装源文件和文档

类库源文件在JDK中以压缩文件lib/src.zip的形式发布，解压缩这个文件来得到源代码。为此只需完成以下步骤：

  1. 找到 JDK 安装目录下的 lib/src.zip 文件。
  2. 解压缩该文件到任意目录，例如 C:\java\src。
  3. 在 IDE 中配置源代码路径，通常在项目设置中添加源代码目录.

文档包含在独立于JDK的一个压缩文件中，可以从 Oracle JDK 文档下载页面 获取。

使用命令行工具

Java 提供了一系列命令行工具来编译、运行和管理 Java 程序。

  • 编译 Java 程序：使用 javac 命令编译 .java 文件。

  • 运行 Java 程序：使用 java 命令运行编译后的 .class 文件。

    使用IDE

集成开发环境（IDE）可以大大简化Java开发过程。常用的Java IDE包括：

  • IntelliJ IDEA：功能强大，支持多种插件，适合大型项目。
  • Eclipse：开源，社区活跃，插件丰富。
  • NetBeans：Oracle官方支持，易于上手。
  • Visual Studio Code：轻量级，支持Java插件。
  • BlueJ：适合初学者，界面友好。

JShell

Java9引入了另一种使用Java的方法。JShell程序提供了一个“读取-评估-打印循环’(Read-Evaluate-PrintLoop,REPL)。键入一个Java表达式，JShell会评估输入,打印结果，并等待下一个输入。

JShell可以用来测试Java代码片段、学习Java语法或快速验证想法。

Jenv

Jenv是一个用于管理Java版本的工具，类似于Python的pyenv。它可以帮助你在不同的Java版本之间切换，特别是在开发需要特定Java版本的项目时非常有用。
