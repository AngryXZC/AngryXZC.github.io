Vue2项目配置,Vue.js项目实战第4章读书笔记

vue-template-compiler

  • Vue 2 中用于编译 .vue 文件模板的工具。
  • 版本必须和 vue 主版本严格一致（比如都是 2.6.14）。
  • 查看方式：
    • package.json 中 vue-template-compiler 字段
    • 命令行：npm list vue-template-compiler
  • Vue 3 不再使用，改用 @vue/compiler-sfc。

SFC（Single File Component）

  • Vue 组件的单文件格式，.vue 文件。
  • 包含 <template>, <script>, <style> 三大块。
  • Vue2 使用 vue-template-compiler 编译，Vue3 使用 @vue/compiler-sfc。
  • 通过构建工具（Webpack + vue-loader）转换成 JS 模块。

Pug 模板引擎

  • 简洁的 HTML 模板语言，缩进表示层级，省略闭合标签。
  • 在 Vue <template lang="pug"> 中使用。
  • 需要安装 pug 和 pug-plain-loader。

JSX

  • JavaScript 的语法扩展，可以在 JS 里写类似 HTML 的结构。
  • Vue2 可以使用 JSX 写 render 函数，需要 Babel 插件 @vue/babel-preset-jsx 支持。

PostCSS

  • 一个处理 CSS 的工具平台，通过插件实现自动加前缀、压缩、未来语法支持等。
  • 常用插件有 autoprefixer, postcss-preset-env, cssnano。
  • Vue CLI 项目默认集成 PostCSS。

CSS 预处理器：SASS / Less / Stylus

名称 特点
SASS 功能强大，支持变量、嵌套、继承、函数，语法分 sass 和 scss 两种，Vue 中常用 SCSS。
Less 语法简单，变量用 @，支持混合、运算，常见于一些中小项目。
Stylus 语法自由度最高，可省略括号和分号，支持条件语句和循环，适合喜欢灵活写法的开发者。
  • 在 Vue 中使用，需安装对应 loader，如 sass-loader、less-loader、stylus-loader。
