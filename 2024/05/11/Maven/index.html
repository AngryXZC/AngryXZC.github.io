еҸ‘иЎЁдәҺ 2024-05-11 еҲҶзұ»дәҺ Maven йҳ…иҜ»ж¬Ўж•°пјҡ жң¬ж–Үеӯ—ж•°пјҡ 458 йҳ…иҜ»ж—¶й•ҝ ≈ 2 еҲҶй’ҹ
MavenеҹәзЎҖжҰӮеҝөгҖҒе®үиЈ…гҖҒзҺҜеўғеҸҳйҮҸй…ҚзҪ®

Maven еҹәзЎҖжҰӮеҝө

  1. Projectпјҡд»»дҪ•дҪ жғіжһ„е»әзҡ„дәӢеҠЎпјҢMaven йғҪеҸҜд»Ҙи®Өдёәе®ғд»¬жҳҜвјҜзЁӢгҖӮиҝҷдәӣвјҜ д№үдёәвјҜзЁӢеҜ№иұЎжЁЎеһӢпјҲProject Object Model, POMпјүгҖӮвјҖдёӘвјҜзЁӢеҸҜд»Ҙдҫқиө–е…¶д»–зҡ„вјҜ зЁӢпјҢвјҖдёӘвјҜзЁӢд№ҹеҸҜд»Ҙз”ұеӨҡдёӘвјҰвјҜзЁӢжһ„жҲҗгҖӮ
  2. POM:POM пјҲpom.xmlпјүжҳҜ Maven зҡ„ж ёвјјвҪӮд»¶пјҢе®ғжҳҜжҢҮвҪ° Maven еҰӮдҪ•вјҜдҪңзҡ„е…ғ ж•°жҚ®вҪӮд»¶пјҢзұ»дјјдәҺ Ant дёӯзҡ„build.xmlвҪӮд»¶гҖӮPOM вҪӮд»¶дҪҚдәҺжҜҸдёӘвјҜзЁӢзҡ„ж №вҪ¬еҪ•дёӯгҖӮ
  3. GroupId:GroupldжҳҜвјҖдёӘвјҜзЁӢеңЁе…ЁеұҖдёӯзҡ„е”ҜвјҖж ҮиҜҶз¬ҰпјҢвјҖиҲ¬ең°пјҢе®ғе°ұжҳҜвјҜзЁӢеҗҚгҖӮ Groupld жңүеҲ©дәҺдҪҝвҪӨвјҖдёӘе®Ңе…Ёзҡ„еҢ…еҗҚе°ҶвјҖдёӘвјҜзЁӢд»Һе…¶д»–жңүзұ»дјјеҗҚз§°зҡ„вјҜзЁӢдёӯеҢәеҲ«еҮәжқҘгҖӮ
  4. ArtifactпјҡдёӯвҪӮеҗҚдёәвҖңжһ„д»¶вҖқпјҢжҳҜвјҜзЁӢе°ҶиҰҒдә§вҪЈжҲ–йңҖиҰҒдҪҝвҪӨзҡ„вҪӮд»¶пјҢе®ғеҸҜд»ҘжҳҜjar вҪӮд»¶гҖҒжәҗвҪӮд»¶гҖҒвјҶиҝӣеҲ¶вҪӮд»¶гҖҒ.war вҪӮд»¶пјҢз”ҡвҫ„жҳҜ.pom вҪӮд»¶гҖӮжҜҸдёӘ Artifact йғҪз”ұ Groupld е’Ң Artifactld з»„еҗҲзҡ„ж ҮиҜҶз¬Ұе”ҜвјҖиҜҶеҲ«гҖӮйңҖиҰҒиў«дҪҝвҪӨзҡ„ Artifact йғҪиҰҒж”ҫеңЁд»“ еә“пјҲи§Ғ RepositoryпјүдёӯпјҢеҗҰеҲҷ Maven вҪҶжі•жүҫеҲ°е®ғд»¬гҖӮ
  5. DependencyпјҡдёәдәҶиғҪеӨҹжһ„е»әжҲ–иҝҗвҫҸпјҢвјҖдёӘе…ёеһӢзҡ„Java вјҜзЁӢдјҡдҫқиө–е…¶д»–зҡ„еҢ…гҖӮеңЁ Maven дёӯпјҢиҝҷдәӣиў«дҫқиө–зҡ„еҢ…е°ұиў«з§°дёә DependencyгҖӮDependency вјҖиҲ¬жҳҜе…¶д»–вјҜзЁӢзҡ„ ArtifactгҖӮ
  6. Plug-inпјҡ еҸҜд»ҘиҜҙ Maven е°ұжҳҜвјҖе ҶжҸ’д»¶зҡ„йӣҶеҗҲпјҢе®ғзҡ„жҜҸвјҖдёӘеҠҹиғҪйғҪжҳҜз”ұжҸ’д»¶е®ҢжҲҗзҡ„гҖӮжҸ’д»¶жҸҗдҫӣgoalпјҲзұ»дјјдәҺ Ant дёӯзҡ„ targetпјүпјҢе№¶ж №жҚ®еңЁPOM дёӯжүҫеҲ°зҡ„е…ғж•°жҚ®еҺ» е®ҢжҲҗвјҜдҪңгҖӮдё»иҰҒзҡ„ MavenжҸ’д»¶жҳҜз”ұ Java зј–еҶҷвҪҪжҲҗзҡ„пјҢеҗҢж—¶вҪҖжҢҒвҪӨ Beanshell жҲ– Antи„ҡжң¬зј–еҶҷзҡ„жҸ’д»¶гҖӮ
  7. Repositoryпјҡд»“еә“пјҢеҚіж”ҫзҪ® Artifact зҡ„ең°вҪ…пјҢжңүдёӯеӨ®д»“еә“гҖҒе…¬е…ұд»“еә“гҖҒз§Ғжңүд»“еә“еҸҠжң¬ең°д»“еә“д№ӢеҲҶгҖӮдёәдәҶжҸҗвҫј Artifactзҡ„дёӢиҪҪйҖҹеәҰпјҢвјҖиҲ¬жғ…еҶөдёӢпјҢе…¬еҸёжҲ–ејҖеҸ‘иҖ…з»„з»ҮйғҪйңҖиҰҒйғЁзҪІвјҖдёӘз§Ғжңүд»“еә“пјҢеҸҜдҪҝвҪӨ Nexus еҲӣе»ә Mavenз§Ғжңүд»“еә“гҖӮ
